## Objetivo
Retornar um array listando a Instituição ou Nome da Capacitação, Latitude e Longitude, o resultado deve ser um array como abaixo:

~~~php
array (
    ['Capacitação TESTE',	'4.60314',	'-60.1815'],
    ['Instituição TESTE',	'-27.5861254',	'-48.5209025']
);
~~~
